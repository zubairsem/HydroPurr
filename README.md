# HydroPurr: Smart Cat Water Dispenser
## Project Overview
HydroPurr is an IoT-based Smart Cat Water Dispenser that helps to reduce electric comsumption by automatically turn on and off the actuator based on surrounding condition and motion.
## Sensors
1. DS18B20 Waterproof Temperature Sensor - monitor water temperature
2. HC-SR501 PIR Motion Sensor - detect cat motion
3. LDR Sensor - detect light exposure
4. Water Level Float Switch - detect water level in the container
## Actuators
1. Buzzer - low water level alarm
2. Pump - pump water
3. LED - lighting up the container area
4. OLED - display data
## Features
1. Temperature monitoring
2. Light exposure monitoring
3. Motion detection
4. Automatic system
5. Remote mobile control
